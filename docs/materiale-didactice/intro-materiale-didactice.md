# I.1. Materialele didactice din Centrum



:::info[Ce vă oferă Centrum ?]


**Centrum** vă pune la dispoziție o gamă largă de materiale didactice organizate pe următoarele categorii:

- Materiale didactice pentru învățământul preșcolar
- Materiale didactice pentru învățământul primar (Clasele I-IV)
- Materiale didactice pentru învățământul gimnazial (Clasele V-VIII)
- Materiale didactice pentru învățământul liceal (Clasele IX-XII)
- Materiale didactice pentru facultate


:::
