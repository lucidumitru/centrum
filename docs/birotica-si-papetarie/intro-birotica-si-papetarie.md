# I.3. Birotică și papetărie



:::info[Ce produse de birotică și papetărie vă oferă Centrum?]


**Centrum** vă pune la dispoziție o gamă largă de produse de birotică și papetărie organizate pe următoarele categorii:


I.3.1. Birotică piese mari (birouri, scaune, dulapuri)
1.3.2. Birotică piese mici (bibiliorafturi, dosare, markere, separatoare, capsatoare)
I.3.3. Produse de papetărie (creioane, radiere, pixuri, stilouri, caiete, blocuri de desen)


:::
