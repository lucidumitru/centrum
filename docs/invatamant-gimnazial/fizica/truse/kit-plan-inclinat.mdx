**I. COMPONENTE**

- Bază din lemn pentru stabilitate

- Plan înclinat reglabil (placă mobilă din lemn)

- Suport vertical cu scală gradată (0–45°)

- Mecanism de reglare a unghiului (articulație + șurub)

- Fir cu greutate (plumb) pentru verificarea verticalității

- Plăcuțe de test din materiale diferite (lemn, material alb, material negru)

- Disc cu fir (pentru experimente suplimentare de mișcare)


<br></br>
<br></br>



**II. EXPLICAȚII TEORETICE**


Planul înclinat este o dispozitiv care permite studiul mișcării corpurilor sub acțiunea gravitației. Pe un plan înclinat, forța gravitațională se descompune în două componente: una paralelă cu planul (responsabilă de mișcare) și una perpendiculară (forța de reacțiune).

Frecarea este forța care se opune mișcării și depinde de natura suprafețelor. Unghiul la care un corp începe să alunece permite determinarea coeficientului de frecare.
