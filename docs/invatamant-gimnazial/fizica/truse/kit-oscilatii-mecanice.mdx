**II. EXPLICAȚII TEORETICE**


Acest kit permite studiul oscilațiilor mecanice, în special a pendulului simplu. Un pendul este un sistem format dintr-o masă suspendată care oscilează sub acțiunea gravitației.

Perioada pendulului este dată de relația: T = 2π√(l/g), unde l este lungimea firului, iar g accelerația gravitațională.

Oscilațiile pot fi studiate în funcție de lungime, masă și amplitudine.
