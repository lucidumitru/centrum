I. PREZENTAREA GENERALĂ A DISPOZITIVULUI

**Planul înclinat** este un dispozitiv care permite studiul mișcării corpurilor sub acțiunea gravitației.



<br></br>
<br></br>




**II. COMPONENTE**

- Plan înclinat din lemn

- Suport din lemn cu cârlig pentru mase marcate

- Corp paralelipipedic din lemn








<br></br>
<br></br>


**III. EXPLICAȚII TEORETICE**


Pe un plan înclinat, forța gravitațională se descompune în două componente: una paralelă cu planul (responsabilă de mișcare) și una perpendiculară (forța de reacțiune).

Frecarea este forța care se opune mișcării și depinde de natura suprafețelor.
