**I. COMPONENTE**


- Două discuri din aluminiu (d ≈ 270 mm) cu acoperire anti corozivă

- Suporturi izolatoare pentru fixare verticală

- Sistem de reglare a distanței între discuri (distanță variabilă până la 0)

- Electrozi centrali pentru conectare la sursă / electrometru

- Placă dielectrică din plexiglas (~220 mm)




<br></br>
<br></br>



**II. EXPLICAȚII TEORETICE**


**Condensatorul plan** este un dispozitiv care stochează energie electrică sub formă de câmp electric. Este format din două plăci conductoare paralele între care există un dielectric (aer sau alt material).


Capacitatea electrică C a unui condensator este raportul constant dintre sarcina electrică Q de pe armături și tensiunea electrică aplicată între armături.

🔦 Observație

Într-un circuit de curent continuu, condensatorul întrerupe circuitul.<br />
Într-un circuit de curent alternativ, condensatorul rând pe rând se încarcă și se descarcă, lăsând să treacă curentul electric prin el. <br />
Câmpul electric dintre plăci este aproximativ uniform și liniile de câmp sunt paralele. <br />
