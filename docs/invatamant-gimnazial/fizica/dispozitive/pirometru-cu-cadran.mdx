**II. EXPLICAȚII TEORETICE**


**Pirometru cu cadran** se folosește pentru a evidenția dilatarea diferită a tijelor metalice, în funcție de natura acestora. Elevii vor constata că la aceeași lungime a tijei și aceeași încălzire, cele 3 metale se dilată diferit. Aluminiul cel mai mult, apoi cuprul și cel mai puțin fierul


La aceeași variație de temperatură unele corpuri se dilată mai mult, altele mai puțin. Fiecare material (substanță) este caracterizat printr-o mărime fizică numită coeficient de dilatare termică liniară, notată cu α.

l = l<sub>0</sub> ( 1 + α ∙ Δt ) <br />
l = lungimea barei dilatată (contractată) <br />
l<sub>0</sub> = lungimea barei înainte de încălzire <br />
α = coeficientul de dilatare termică liniară a materialului din care este confecționată bara <br />
Δt = variația temperaturii barei <br />
