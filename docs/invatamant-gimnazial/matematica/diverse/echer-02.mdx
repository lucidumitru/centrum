**I. PREZENTARE PRODUS**



**Echerul** este un instrument esențial pentru trasarea, măsurarea și verificarea unghiurilor drepte (de 90°), a perpendicularității a două drepte în plan, la trasarea liniilor paralele, unghiurilor de 45° etc. Este un instrument utilizat frecvent de elevi la orele de matematică, fizică, geografie, desen. Echerul mai este utilizat în tâmplărie, construcții și desen tehnic.

O catetă a echerului este gradată în centimetri (0-9,9 cm), iar cealaltă catetă este gradată în inch (0-3,85 inch). Are forma unui triunghi dreptunghic isoscel, cu un unghi de 90°, celelalte unghiuri fiind de 45°. Este realizat din plastic transparent.
