**I. PREZENTARE PRODUS**



**Echerul** este un instrument esențial pentru trasarea, măsurarea și verificarea unghiurilor drepte (de 90°), a perpendicularității a două drepte în plan, la trasarea liniilor paralele, unghiurilor de 60°/30° etc. Este un instrument utilizat frecvent de elevi la orele de matematică, fizică, geografie, desen. Echerul mai este utilizat în tâmplărie, construcții și desen tehnic.

O catetă a echerului este gradată în centimetri (0-13 cm), iar cealaltă catetă este gradată în inch (0-3 inch). Are forma unui triunghi dreptunghic, cu un unghi de 90°, celelalte unghiuri de 60°, respectiv 30°. Este realizat din plastic transparent.
